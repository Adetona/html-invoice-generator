What can I do with this template? ●   Write professionally looking invoice in a matter of seconds. ○   Use any language or currency. ○   Add your own logo with simple drag and drop. ○   Easily add, remove, and reorder item rows. ○   Add taxes and discounts. ○   Configure item column visibility. ○   Auto calculate subtotals and totals as you type. ○   Save your default company data for future re-use. ●   Restyle it by using simple HTML & CSS. ●   Directly print it out. How can I save my default company data for future re-use? This data is stored in the data.js file so you can easily edit it with any text editor. To save your logo, you'll need to supply base64 encoded data URI of it (simply put - text representation of the image). No matter how scary this might sound, the only thing you need to do is drag and drop your logo to this template and then click on it. A popup window will show with generated base64 data URI (long string of characters) of the logo, which you need to copy and paste it in the data.js file (into the value of the {company_logo} property). That's all. The data and the structure of the data.js file is quite simple and self-explanatory. Any questions whatsoever, feel free to hit us up at support@invoicebus.com - you'll be hearing from us very soon! How to save the invoice? This one is a bit tricky. Due to the limitation of how browsers work with local html files, we haven’t figured out yet how to store any dynamic data you enter, such as items, prices, taxes, etc. However, you have few other alternatives -   Use the embedded PDF printer in Google Chrome. -   Use free PDF print driver such as Foxit Reader PDF Printer or BullZip. -   Use the green "Save" button that will take you to our online invoicing service Invoicebus (which happens to be a great tool by the way) and securely store it into your personal account. While the first two options will generate static PDF version of the invoice (which cannot be modified), the third option will generate live online invoice (like this) which can be edited on-the-fly, sent by email, tracked, and paid directly. You will also be able to re-use every item, client, and tax so you can write invoices goddamn fast in the future. How to restyle the template? As the template is completely separate from its logic, you can easily restyle it by using simple HTML and CSS techniques. However, there are few rules which are consider to be good practice when making structural changes (deleting or rearranging elements) of the DOM, so have in mind to follow this guide. Do I have to be connected to internet to use it? Yes, but isn't necessary. In the background the template links to remote JavaScript file (generator.min.js) located on our server from which it pulls all the logic. If for some reason you don't have internet access on your computer, you can download the script locally, and modify the template.html file to point to the local version of the script. We generally encourage you to use the remote script because we're constantly improving it, so any recent updates immediately reflect into the template you're using. And what about the fonts used? The template uses Google Fonts as they are free and open to everyone. If you want to use the template offline you’ll have to download and install the fonts on your computer (SkyFonts is a simple tool that helps you do this easily). Can I use the JavaScript source code?
